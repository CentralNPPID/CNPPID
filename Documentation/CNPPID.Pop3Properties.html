Pop3 Properties

The properties of the Pop3 class are listed below. For a complete list of Pop3 class members, see the Pop3 Members topic.

Public Instance Properties

AttachmentDirectory This is the directory where the attachments are stored if SaveAttachments is True.
Cancel Set this to True if you want the mail control to stop what it's doing. It is reset to False when the control checks mail again.
CurrentState What the control is currently doing.
DownloadHeadersOnly Set this to True before checking the mail if you only want the headers (From, To, Date), etc. and not the message body or the attachments.
NumMessages How many messages the control retrieved from the mail server.
Password Set this before checking the mail so the control can log into the server to check the mail.
SaveAttachments Set this to True if you want the attachments automatically saved to disk. Don't forget to set the AttachmentDirectory attribute as well.
ServerName The name of the server to connect to and retrieve the emails from.
UserName This is the account name to use when logging into the mail server.
