AttachmentDirectory This is the directory where the attachments are stored if SaveAttachments is True.
Cancel Set this to True if you want the mail control to stop what it's doing. It is reset to False when the control checks mail again.

Public Instance Methods

Connect Try to connect to the mail server with the previously supplied user name and password. You need to define the user name, password and server name before using this function. Blank passwords are allowed but not blank user IDs or server names. NumMessages is set once the connection is made. Returns True if it was able to connect or False if not.
CreateFromText Break apart strMessage into the header and body sections and create a mail message from it. If SaveAttachmentsToDisk is True, the attachments are saved to that directory. strMessage is the string used as a source. Returns a Pop3Message with all of the data filled in.
DecodeBase64 Save an attachment encoded as Base64 to a file on disk. strMsg is the attachment to decode. strPath is the full path, including filename, of the file to use when saving to disk.

Returns True if we were successfully able to save the attachment.

Returns False if an error occurred.

DeleteByGUID Delete the message with GUID of strGUID. Goes through the list of messages already downloaded to get the index of the message with the same GUID. This gives us a starting point to start querying the server for the index. Once we find the index, delete it and return True if we succeeeded or False if not. strGUID is the GUID of the message to delete.

Returns True if the message is succesfully deleted.

Returns False if the message is not deleted.

DeleteMessage Delete the message with index number msgIndex. NOTE that this is the 1-based index the mail server uses, not the 0-based index that the class uses. Returns True if it was able to delete the message, False if not. This assumes we are already connected to the mail server. msgIndex is the index on the mail server.

Returns True if the message was successfully deleted.

Returns False if the message was not successfully deleted.

DownloadMessages Download the messages from the server and store them in the Messages() array. The user can use the GetMessageByIndex() function to retrieve a particular message from the array. Sets intNumMessages to the proper amount.

Returns True if the class is able to retrieve messages.

False if an error occurs.
