clsKeyStrokes.KeyPressDecimal Method 

Check to see if the key pressed is a number (0-9) or if it is a period. e is the KeyPressEventArgs sent into Box's KeyPressed event. Box is either a TextBox or ComboBox. It is needed so we can get the Text property and check for the presence of a decimal.

This returns True if the key is valid.

Returns False and sets the e.handled property if the key press is not valid.
