clsKeyStrokes.KeyPressInteger Method 

Evaluate a key press from the user to see if it was an integer (0-9). e is the KeyPressEventArgs sent to the calling sub.

Returns True if the key stroke is valid.

Returns False and sets the e.Handled property to True if the key is invalid.
