Pop3.DeleteMessage Method 

Delete the message with index number msgIndex. NOTE that this is the 1-based index the mail server uses, not the 0-based index that the class uses. Returns True if it was able to delete the message, False if not. This assumes we are already connected to the mail server. msgIndex is the index on the mail server.

Returns True if the message was successfully deleted.

Returns False if the message was not successfully deleted.
