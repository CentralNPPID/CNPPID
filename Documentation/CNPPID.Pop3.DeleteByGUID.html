Pop3.DeleteByGUID Method 

Delete the message with GUID of strGUID. Goes through the list of messages already downloaded to get the index of the message with the same GUID. This gives us a starting point to start querying the server for the index. Once we find the index, delete it and return True if we succeeeded or False if not. strGUID is the GUID of the message to delete.

Returns True if the message is succesfully deleted.

Returns False if the message is not deleted.
