clsKeyStrokes.KeyPressAlpha Method 

This tests the key pressed to see if it's in the valid list of keys. strValidKeys can be any string that contains the valid characters, like "123AB" or ",.lad8D".

If the key stroke is not in the valid list, this sets e.Handled property to True and returns False.

Otherwise, it leaves the Handled property alone and returns True.
