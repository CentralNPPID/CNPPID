SettingsFile.AddSetting Method 

Add a setting with key strK and data strD and an optional description of strDesc. Increment intNumSettings. strK is the key to use. It must be unique or you will have problems getting the values out later. strD is the data for the key. strDesc is the description for this item. It is optional.

Returns True if we were able to add the setting.

Returns False if not.
