clsCountingList.BinarySearch Method 

Do a binary search of the array and look for strKey. This assumes the list is sorted. If it isn't, use Sort(). The search is not case sensitive. strKey is the key to look for.

Returns the index of strKey in the array if strKey is in there.

Returns -1 if it's not found.
