KeyPressAlpha This tests the key pressed to see if it's in the valid list of keys. strValidKeys can be any string that contains the valid characters, like "123AB" or ",.lad8D".

If the key stroke is not in the valid list, this sets e.Handled property to True and returns False.

KeyPressDecimal Check to see if the key pressed is a number (0-9) or if it is a period. e is the KeyPressEventArgs sent into Box's KeyPressed event. Box is either a TextBox or ComboBox. It is needed so we can get the Text property and check for the presence of a decimal.

This returns True if the key is valid.

Returns False and sets the e.handled property if the key press is not valid.
